# city-management-system-core

## Prerequisites

Java 17.0.9

Gradle 8.4

## Start

1. Clone the repository - `git clone https://github.com/spacheliyska/city-management-system-core.git`
2. Navigate to the project folder - `cd city-management-system-core`
3. Build the project
```bash
./gradlew clean build
```
4. Start the project
```bash
./gradlew bootRun
```
